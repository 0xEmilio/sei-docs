# Setting up a Wallet

## Installing a Wallet

Sei supports multiple native wallets.

- [Compass](https://compasswallet.io/)
- [Leap](https://leapwallet.io/)
- [Fin](https://finwallet.com/)

For more wallet options, please visit our [registry](https://github.com/sei-protocol/chain-registry/blob/main/wallets.json).

### Import Manually

On the wallets listed above, they should be automatically be configured to support Sei.
However, other unsupported wallets may require chain information to support Sei. Some of the following sample config might be helpful here:
